うたたね書庫

女性のための泊まれる図書館

concept　ブックホテルとは？

目まぐるしい日常を離れて、気が済むまで本に埋もれる。

たまにはそんな一日があってもいいと思いませんか？

当ホテルでは館内のあらゆるところに本棚が置かれ、並んでいる本を手にとって読めるようになっています。

読書室や喫茶室で読むのも、もちろんお部屋に戻って読むのも自由です。

女性専用のブックホテルだからメイクや服装に気をつかう必要もありません。

読書のあいまに珈琲を飲んだり、ちょっとうたたねしてひと休みしたり。

ゆっくりとくつろぎながら本が読める環境を提供いたします。

how to use　本の利用方法

  • 本棚の本は、館内の好きな場所でお読みいただけます。

  • 読書室に置いてある貸出用ブックカバーをかけてご利用ください。

  • 読み終わった本は、返却ボックスへ。

※読書中、フタのついた飲み物以外の飲食はご遠慮いただきますようお願いいたします。

stay　客室紹介

お客様が宿泊するお部屋です。

シングルとセミダブルの二種類から選ぶことができます。

本を読みながら眠ってしまった場合にも体を傷めないよう、ベッドはシモンズ製のものをご用意しました。

読書に集中できる環境づくりのためテレビは備わっておりません。

館内で着用できる館内着や各種アメニティもご用意しています。

FreeWifiを利用する際は入館時にお渡しするパスワードをご使用ください。

room　館内案内

読書室 library

本と本棚が集まっているお部屋です。

いろんな形の座り心地の良い椅子とカーペットの上にクッションを並べたラウンジスペースがあり、集中して読書ができるようになっています。

今月の特集本が並ぶ本棚や外国の絵本を並べた本棚など、見て楽しめる本棚もご用意しています。

カウンターには話を聞いておすすめの本を選んでくれる本のコンシェルジュがおりますので、本選びでお困りの際にはご相談ください。

喫茶室 cafe

フタつきの飲み物を提供しているお部屋です。

珈琲、紅茶、ルイボスティー、ジュース、緑茶など、お好きな飲み物を何度でも飲むことができます。

ご利用の際にはカウンターにいるスタッフにお声がけください。

その他にも、本を読んだりひと休みするのにご活用ください。

食事室 dining

食事を提供しているお部屋です。

夕食の時間は18:00と19:00のどちらかから選ぶことができます。

朝食は8:00～11:00まで提供しております。お好きな時間にご利用ください。

また、ウエルカムスイーツとしてチョコレートのカップケーキをご用意しています。
